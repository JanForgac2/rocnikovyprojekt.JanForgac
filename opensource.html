Slobodný a otvorený softvér

Slobodný a otvorený softvér sa nazýva aj open source softvér. Pri tomto type softvéru je verejne dostupný zdrojový kód programu. Používatelia môžu program upravovať, vylepšovať a ďalej šíriť.

Na vývoji open source programov často pracujú programátori z celého sveta. Takýto softvér býva pravidelne aktualizovaný a rýchlo opravovaný.

Open source softvér je populárny hlavne preto, že býva bezpečný, bezplatný a ponúka veľkú slobodu používateľom.

Výhody open source softvéru

  • bezplatné používanie
  • možnosť úpravy programu
  • vysoká bezpečnosť
  • rýchle opravy chýb

Nevýhody open source softvéru

  • niektoré programy sú zložitejšie
  • slabšia technická podpora
  • nie vždy moderný dizajn

Najznámejším open source operačným systémom je Linux. Medzi ďalšie programy patrí LibreOffice alebo Mozilla Firefox.
